## Design Principles

### Core Design Values

1. **Clarity** - Clean, intuitive interfaces with clear visual hierarchy
2. **Efficiency** - Minimize clicks and cognitive load for common tasks
3. **Accessibility** - WCAG 2.1 Level AA compliant
4. **Consistency** - Uniform design patterns across all screens
5. **Feedback** - Clear feedback for all user actions

### Design Goals

- **For Production Managers:** Provide comprehensive overview, quick access to critical information, and efficient workflow management
- **For Production Admin:** Simple, mobile-friendly interface for fast data entry with clear validation feedback
- **For Both:** Reduce training time, minimize errors, and improve productivity

## Typography

### Font Family

- **Primary Font:** 'Roboto', 'Segoe UI', sans-serif
- **Monospace Font:** 'Roboto Mono', monospace (for codes and numbers)

### Font Sizes

| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 - Page Title | 32px | 500 | 1.2 |
| H2 - Section Title | 24px | 500 | 1.3 |
| H3 - Card Title | 20px | 500 | 1.4 |
| H4 - Component Title | 16px | 500 | 1.5 |
| Body Text | 14px | 400 | 1.5 |
| Small Text | 12px | 400 | 1.4 |
| Caption | 11px | 400 | 1.3 |

### Font Weights

- 300 - Light (labels, captions)
- 400 - Regular (body text)
- 500 - Medium (headings, buttons)
- 700 - Bold (emphasis, strong text)

   - Primary: Click to submit/confirm
   - Secondary: Click to cancel/reset
   - Icon: Click for quick action

   - Real-time validation on blur
   - Clear error messages
   - Success feedback on submit

   - Progress bar for long operations
   - Skeleton screens for data loading

   - Graceful degradation
   - Retry options
